Python is a multi-paradigm, object-oriented and structured programming language.
Python is an example of a FLOSS (Free/Libre and Open Source Software).
Due to its open-source nature, Python has been ported to many platforms.
You can use Python on Linux, Windows, FreeBSD, Macintosh, Solaris, OS/2, Amiga, AROS, AS/400, BeOS, OS/390, z/OS, Palm OS, QNX, VMS, Psion, Acorn RISC OS, VxWorks, PlayStation, Sharp Zaurus, Windows CE and even PocketPC!
